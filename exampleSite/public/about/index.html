About

This is a demonstration site for the Darklane Hugo theme. Darklane is a clean, dark-themed blog template designed for technical writing and documentation.

Theme Features

  • Dark color scheme - Easy on the eyes with carefully chosen colors
  • Syntax highlighting - Beautiful code blocks with multiple language support
  • Custom shortcodes - Alerts, highlights, collapsible sections, tabs, and more
  • LaTeX support - Write mathematical expressions inline or as display equations
  • Responsive design - Works great on desktop and mobile
  • CSS variables - Easy color customization through a single configuration

Getting Started

To use this theme for your own site, visit the GitHub repository for installation instructions and documentation.
